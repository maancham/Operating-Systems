## General info
Course projects related to Operating Systems.
	
## Technologies
There are 3 projects within this repository which are as follows:
* First phase: C language. A client-server messaging program capable of handling multiple clients at once.
* Second phase: C++. A multi-process program to find records within a text file. Named and unnamed pipes were used to speed up the process of searching and returning results.
* Third phase: C++. A practice of multi-threading and measuring the improvements compared to serial implementation.
